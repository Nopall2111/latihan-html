Program Studi D3 Manajemen Informatika

Program ini menekankan pengelolaan sistem informasi, analisis data, dan pengembangan aplikasi bisnis. Mahasiswa mempelajari desain database, manajemen data, serta pemrograman aplikasi untuk mendukung pengambilan keputusan dalam perusahaan. Lulusan dapat berkarier sebagai analis sistem, administrator database, atau pengembang aplikasi bisnis. Dengan meningkatnya kebutuhan akan pengelolaan data yang efisien, prospek kerja lulusan program ini sangat menjanjikan.
