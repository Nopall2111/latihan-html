Program Studi D4 Teknologi Rekayasa Perangkat Lunak

Program sarjana terapan ini mengajarkan teknik pengembangan perangkat lunak, mulai dari analisis kebutuhan, pemrograman, hingga manajemen proyek IT. Mahasiswa juga mempelajari pengembangan aplikasi berbasis web dan mobile sesuai standar industri.

Dengan kurikulum yang berorientasi pada praktik dan magang di perusahaan IT, lulusan program ini siap berkarier sebagai software engineer, system analyst, web developer, atau mobile application developer.
